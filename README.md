# Анализ туристической индустрии российских отелей

## Обзор проекта

Данный проект направлен на сбор, очистку, анализ и объединение данных из нескольких источников, таких как **OpenTripMap**, **Otello** и **Hotellook**. Основная цель проекта заключается анализе данных отелей, чтобы помочь бизнесу принимать более эффективные решения для получения высоких рейтингов на сайтах.

## Описание данных

### Источники данных

- **OpenTripMap**  
  Набор данных о туристических объектах включает сведения как о достопримечательностях, так и об отелях. Данные собирались через API OpenTripMap, при этом бесплатный API-ключ позволяет выполнять до 5 000 запросов в день и 10 запросов в секунду.

- **Otello**  
   На начальном этапе применяется API для извлечения списка URL-адресов веб-страниц, содержащих информацию об отелях. После формирования списка выполняется парсинг каждой страницы для получения данных: названия, стоимости проживания, рейтинга, адреса и других сопутствующих сведений.

- **Hotellook**  
  Данные о предложениях отелей, включающие цену за ночь, звёздность отеля, рейтинг, доступность номеров и прочие характеристики. Сбор данных осуществлялся через использование API в открытом доступе.

## Выводы проекта

#### 1. Важность удобств и инфраструктуры:

Некоторые услуги (общая ванная, общие номера) имеют отрицательное влияние на рейтинг, большинство слабо влияет, но положительно, некоторые имеют достаточно выраженную положительную связь (помощь маломобильным людям, бассейн). При этом нет выраженной взаимосвязи между частотой встречаемости услуги и влиянием на рейтинг.

#### 2. Локация и ее влияние на рейтинг

Близость к центру – преимущество: рейтинг выше у отелей в центральных или туристических (для приморских городов) районах. При этом самые низкие рейтинги у обьектов в жилых районах. Обьекты на окраине могут быть высоко оценены, если предоставляют услуги или туристический интерес.

#### 3. Рейтинг ≠ качество

Рейтинг отеля не всегда зависит от количества звезд. 5 звездочные отели имеют разбросанные оценки, что подтверждает гипотезу о завышенных ожиданиях клиентов. Дорогие отели могут получать низкие оценки – высокая цена не гарантирует премиального сервиса. Можно найти дорогие мало звездочные отели и относительно доступные 5 звездочные отели.

## Рекомендации для бизнеса

### Оптимизация набора услуг:

- Инвестировать в сервисы с выраженным положительным влиянием.
- Пересмотреть использование услуг с отрицательным влиянием и заменить их более качественными решениями.
- Сфокусироваться на качестве, а не на количестве предлагаемых удобств.

### Улучшение локационной привлекательности:

- Подчёркивать территориальные преимущества отелей, расположенных в центре, и концентрировать для них маркетинг.
- Для объектов в жилых районах разработать дополнительные услуги, чтобы компенсировать недостатки расположения.
- Для отелей на окраинах использовать уникальные предложения для привлечения клиентов.

### Контроль за ожиданиями и ценообразованием:

- В 5-звездочных отелях уделять особое внимание управлению ожиданиями клиентов через сбор обратной связи и оперативное реагирование на отзывы.
- Обеспечить соответствие логики «высокая цена = реальное качество сервиса».
